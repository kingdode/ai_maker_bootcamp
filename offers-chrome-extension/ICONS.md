# Icon Files Required

The extension manifest references icon files that need to be added:

- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)  
- `icon128.png` (128x128 pixels)

## Quick Solution

You can create simple placeholder icons or use any 16x16, 48x48, and 128x128 pixel PNG images.

Alternatively, you can temporarily remove the icon references from `manifest.json` if you just want to test the extension functionality.

## Creating Icons

You can:
1. Use any image editor to create simple icons
2. Use online icon generators
3. Use placeholder images for testing

The extension will work without icons, but Chrome will show a default puzzle piece icon instead.
